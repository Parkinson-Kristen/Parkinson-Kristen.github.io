A Basic HTML Page

With Brackets or your favorite text editor installed and running, you are ready to create your first basic web page. You will be creating this page and supporting folders locally, meaning on your own computer. 

Use the lesson's required readings to help you understand the purpose of the assignment and to provide a guide in completing the assignment. For example, review Dealing with Files to review naming conventions and file references.

Use the Developer Forum to discuss markup development and software issues with your peers. It is OK to post your HTML markup for others to review.

Requirements

  • Create a folder on your computer that will contain this lesson's pages and folders. In the next lesson we will create your official, local cit230 folder so this folder in this lesson is temporary. 
  • Create the following named folders within your lesson folder:
    • images
    • styles or css
    • scripts or js
  • Create a file named basicpage.html. Copy the following HTML markup and content, substituting the generic content with your information:
    • include your name,
    • your current location in terms of a state or country, 
    • how much RAM is on your main computer,
    • a individual photo,
    • and a link to BYU-Idaho online as shown in the image.
